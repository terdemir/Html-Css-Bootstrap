Css bir web sayfasında hangi yöntemler ile kullanılabilir?

  1. Yerel Css
  2. Genel/Dahili Css
  3. Harici/Bağlantılı Css

Yerel Css

Yerel css bir web sayfasında bir html etiketi için özel olarak kullanılır.
Yerel css kullanım örneği bu paragrafın parametresi olarak yer alan style içinde vardır.

Bu paragrafta style parametresi yoktur. Doğal olara herhangi bir biçimlendirme görülmemektedir.

Genel/Dahili Css

Dahili Css yazıldığı sayfanın tamamında kullanılabilir.
Yani bu stiller ile ilgili sayfadaki ilgili etiketlerin hepsinin özellikleri belirlenebilir.
Genel css etiketleri bir web sayfasında head bloğunda style etiketi içine yazılır. Örnek bu sayfanın head bloğunda yer almaktadır. Yazılan özellikler sadece yazılan sayfada etkili olacaktır.

Harici/Bağlantılı Css

Harici css birden fazla sayfada kullanılabilir. Birden çok sayfanın aynı stillere sahip olması istendiğinde kullanılır.
Harici css oluşturmak için .css uzantılı ayrı bir belge gereklidir. Bu belge head bloğunda link etiketi ile sayfaya dahil edilir. Örneği bu sayfanın head bloğunda vardır.
Örneğin bu sayfanın h1 etiketleri harici css ile tanımlanmıştır.

Seçiciler

Seçiciler bir html belgesinde herhangi bir html elemanını ya da birden fazla html elemanını seçerek özelliklerini biçimlendirmemizi sağlar.

Sınıf (class) seçicisi

Sınıf seçicisi birden çok html elemanını biçimlendirmek istediğimizde kullanılır
.sınıfadi şeklinde css içinde tanımlanır. Kullanmak istediğiniz etiket içinde class parametresi ile sınıfadi yazılarak çağırılır. Örnek bu paragrafın yer aldığı p etiketinde vardır.

Id seçicisi

Id seçicisi bir html elemanını özel biçimlendirmek istediğimizde kullanılır.
Css içinde #idadi şeklinde tanımlanır. Etkilenecek olan html etiketi içinde id parametresi ile çağırılarak kullanılır. Örnek bu paragrafın yer aldığı p etiketinde vardır.
