Css bir web sayfasında hangi yöntemler ile kullanılabilir?

  1. Yerel Css
  2. Dahili/Genel Css
  3. Harici/Bağlantılı Css

Yerel Css

Yerel Css bir web sayfasında bir html etiketi için özel olarak kullanılır.
Yerel Css kullanım örneği bu paragrafın parametresi olarak yer alan style içinde vardır.

Bu paragrafta style parametresi yoktur. Doğal olarak herhangi bir biçimlendirme görülmemektedir.

Dahili/Genel Css

Genel Css yazıldığı sayfanın tamamında etkilidir.
Yani bu stiller ile ilgili sayfadaki ilgili etiketlerin hepsinin özelliklerini belirlenebilir.
Genel Css etiketleri head bloğunda style etiketi içine yazılır. Yazılan özellikler sadece yazılan sayfada etkili olacaktır.

Harici/Bağlantılı Css

Harici Css birden fazla sayfada kullanılabilir. Birden çok sayfanın aynı stillere sahip olması istendiğinde kullanılır.
Harici css oluşturmak için .css uzantılı ayrı bir belge oluşturulur. Bu belge Head bloğunda Link etiketi ile sayfada dahil edilir. Örneği bu sayfanın head bloğunda vardır.
Örneğin bu sayfanın h1 etiketleri harici css sayfasında tanımlanmıştır.

Seçiciler

Seçiciler bir html belgesinde bulunan herhangi bir html elemanını veya birden fazla elemanı seçerek özellikleri biçimlendirmemizi sağlar.

Sınıf Seçicisi

Sınıf seçicisi birden çok html elemanını biçimlendirmek istediğimizde kullanılır.
.sınıfadi şeklinde css içinde tanımlanır. Kullanmak istediğiniz etiket içinde class parametresi ile sınıf adı yazılarak kullanılır. Örnek bu paragrafın yer aldığı p etiketinde vardır.

İd Seçicisi

Id seçicisi bir html elemanını özel biçimlendirmek istediğimizde kullanılır.
Css içinde #idadi şeklinde tanımlanır. Etkilenecek olan html etiketi içinde id parametresi ile çağırılarak kullanılır. Örnek bu paragrafın yer aldığı p etiketinde vardır.
